Place Fanatec App XML car list files here to override the auto-detected
files from the Fanatec App installation. Files are named:

  CarsList_iRacing.xml
  ProfileCarsList_iRacing.xml
  CarsList_AC.xml
  ProfileCarsList_AC.xml
  CarsList_ACC.xml
  ProfileCarsList_ACC.xml

These files are found in:
  C:\Program Files\Fanatec\FanatecService\Service\xml\

Copy them here if you want to use custom car lists or if the Fanatec
App is not installed on this PC.

Files in this directory take priority over the Fanatec App install path
and any custom [xml] path set in fanatec-tuner.toml.
